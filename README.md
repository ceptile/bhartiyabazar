# 🏪 BhartiyaBazar

> India's most trusted business search platform — Google + Marketplace + Directory + Trust Engine

## Tech Stack
- **Framework:** Next.js 15 (App Router)
- **Language:** TypeScript
- **Styling:** Tailwind CSS v4 + shadcn/ui
- **Icons:** Lucide React
- **Fonts:** EB Garamond (logo) + Inter + Rajdhani
- **Deployment:** Vercel

## Getting Started
```bash
pnpm install
pnpm dev
```

Open [http://localhost:3000](http://localhost:3000)

## Project Structure
```
src/app/          → Pages (App Router)
src/components/   → Reusable components
src/lib/          → Utilities
src/styles/       → Global styles
```
